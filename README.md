Advent of Code 2023
===================

### How-to

##### For nix users

```
$ nix develop

# running: nimble run aoc2023 DAY_NUMBER input/FILE.txt
$ nimble run aoc2023 1 input/01.txt

# testsuite:
$ nimble test
```

There is also an `.envrc` for [direnv](https://direnv.net) already included.
